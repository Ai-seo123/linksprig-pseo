# LinkSprig WordPress CPT Template Setup Guide

This guide describes how to configure reusable theme templates in WordPress to display the programmatic content and Advanced Custom Fields (ACF) data generated by our pipeline.

---

## Option 1: Custom PHP Theme Templates (Recommended for Speed & SEO)

If you are using a classic theme or child theme, you can create dedicated PHP template files in your theme directory. WordPress will automatically pick them up based on the template hierarchy.

### 1. Comparison Page Template (`single-compare.php`)
Place this file in your active theme's folder (e.g., `wp-content/themes/your-theme/single-compare.php`).

```php
<?php
get_header();

// Fetch ACF Field values
$competitor_name     = get_field('competitor_name');
$competitor_strength = get_field('competitor_strength');
$competitor_weakness = get_field('competitor_weakness');
$ideal_user         = get_field('ideal_user');
$comparison_summary  = get_field('comparison_summary');
$cta_text            = get_field('CTA');

if (have_posts()) : while (have_posts()) : the_post();
?>
<article id="post-<?php the_ID(); ?>" <?php post_class('pseo-compare-article'); ?>>
    <header class="entry-header container">
        <h1 class="entry-title"><?php the_title(); ?></h1>
    </header>

    <div class="entry-content container">
        <!-- Summary Box -->
        <?php if ($comparison_summary) : ?>
            <div class="comparison-summary-box card">
                <h3>Quick Verdict</h3>
                <p><?php echo esc_html($comparison_summary); ?></p>
            </div>
        <?php endif; ?>

        <!-- Core Article Body -->
        <div class="article-body">
            <?php the_content(); ?>
        </div>

        <!-- Competitor Specific Highlights -->
        <div class="competitor-details-grid">
            <div class="strength-card">
                <h3><?php echo esc_html($competitor_name); ?> Strengths</h3>
                <p><?php echo nl2br(esc_html($competitor_strength)); ?></p>
            </div>
            <div class="weakness-card">
                <h3><?php echo esc_html($competitor_name); ?> Weaknesses</h3>
                <p><?php echo nl2br(esc_html($competitor_weakness)); ?></p>
            </div>
        </div>

        <!-- Dynamic CTA -->
        <?php if ($cta_text) : ?>
            <div class="cta-banner text-center">
                <h2>Ready for a better outreach workflow?</h2>
                <p>Move past legacy CRMs. LinkSprig automates prospecting and scales replies.</p>
                <a href="/signup" class="btn btn-primary btn-lg"><?php echo esc_html($cta_text); ?></a>
            </div>
        <?php endif; ?>
    </div>
</article>
<?php
endwhile; endif;

get_footer();
```

### 2. Industry Page Template (`single-industry.php`)
Place in `wp-content/themes/your-theme/single-industry.php`.

```php
<?php
get_header();

$industry_name    = get_field('industry_name');
$seo_challenge    = get_field('SEO_challenge');
$outreach_problem = get_field('outreach_problem');
$relevant_feature = get_field('relevant_feature');
$success_metric   = get_field('success_metric');

if (have_posts()) : while (have_posts()) : the_post();
?>
<article id="post-<?php the_ID(); ?>" <?php post_class('pseo-industry-article'); ?>>
    <header class="entry-header container">
        <h1 class="entry-title"><?php the_title(); ?></h1>
    </header>

    <div class="entry-content container">
        <!-- Industry Stats Callout -->
        <div class="industry-stat-badge">
            <span class="stat-label">Target Metric:</span>
            <span class="stat-value"><?php echo esc_html($success_metric); ?></span>
        </div>

        <div class="article-body">
            <?php the_content(); ?>
        </div>

        <!-- ACF Detail Block -->
        <div class="industry-acf-details">
            <h3>Key Takeaway</h3>
            <p><strong>Primary SEO Challenge:</strong> <?php echo esc_html($seo_challenge); ?></p>
            <p><strong>Common Outreach Bottleneck:</strong> <?php echo esc_html($outreach_problem); ?></p>
            <p><strong>LinkSprig feature utilized:</strong> <?php echo esc_html($relevant_feature); ?></p>
        </div>
    </div>
</article>
<?php
endwhile; endif;

get_footer();
```

## Option 2: Page Builders (Elementor Pro / Divi / Beaver Builder)

If you use a page builder, you can design these templates visually.

### Steps for Elementor Theme Builder:
1. Go to **Templates** -> **Theme Builder** -> **Single Post** -> **Add New**.
2. Set the display conditions to match your custom post type (e.g. `Comparisons` or `Industries`).
3. To display the standard AI content: Add the **Post Content** widget.
4. To display custom ACF fields:
   - Add a **Text Editor** widget.
   - Click the **Dynamic Tags** icon (database cylinder icon) next to the content area.
   - Select **ACF Field** from the dropdown.
   - Click the settings gear next to the field name and choose the specific field (e.g. `competitor_strength`).
5. Set your CTA button link to use the ACF `CTA` text field dynamically or link directly to `/signup`.

## Option 3: Gutenberg Full Site Editing (FSE) block themes

For modern block themes (like Twenty Twenty-Four):
1. In the WordPress admin, navigate to **Appearance** -> **Editor** -> **Templates**.
2. Click the `+` sign and add a new template for a single custom post type (e.g., `Single: Comparison`).
3. Drag in block layouts: **Post Title**, **Post Content**.
4. To display ACF values, install a block-compatible metadata plugin like **ACF Blocks** or use standard Block Bindings (introduced in WordPress 6.5) to bind paragraph blocks directly to custom fields.
